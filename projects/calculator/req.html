Project: Simple Calculator App

Create a calculator app that matches the design mockup below. Use HTML + CSS + JavaScript (no frameworks).

Design Mockup (SVG)

Match this layout: Header + centered calculator card with a display and a 4-column button grid.

1. Required Files

  • index.html (calculator UI)
  • style.css (all styling)
  • script.js (calculator logic)

2. Required UI Elements

  • A calculator display that shows the current number/result
  • A calculator button grid with these buttons:
    • Numbers: 0–9
    • Decimal: .
    • Operators: +, −, ×, ÷
    • Equal: =
    • Clear: C (reset everything)
    • Backspace: ⌫ (delete last digit)
    • Percent: % (convert current number to percent)

3. Required Features (Logic)

  • Clicking number buttons updates the display
  • Clicking an operator stores the operation (+, −, ×, ÷)
  • Clicking = calculates and shows the result
  • C clears all values and resets display to 0
  • ⌫ removes the last digit from the display
  • % changes the current number to: number ÷ 100

4. Error Handling Requirements

  • Division by zero must not crash
  • If user tries: 5 ÷ 0 display must show Error (or a message like Cannot divide by 0)
  • After showing error, pressing C must reset correctly

5. Optional Bonus Features (Extra Points)

  • Keyboard support (typing numbers, Enter for equals)
  • Show the full expression above the main display
  • Button click animation (CSS)
  • Dark mode toggle

6. Design & Technical Rules

  • Follow the mockup layout as closely as possible
  • Use CSS grid for the button layout
  • Must work on mobile (buttons not too small)
  • No frameworks (Bootstrap/Tailwind)
  • No inline styles — all styling in style.css
  • Use semantic HTML where possible

7. Submission Checklist

  • All required buttons exist and work
  • Display updates correctly
  • Operations calculate correctly (+ − × ÷)
  • C / ⌫ / % work correctly
  • No page refresh needed
  • Layout matches mockup

Hints (Open only if needed)

Hint 1: File setup

Create three files and connect them:

  • Link style.css inside <head>
  • Link script.js using defer
Hint 2: HTML structure

Your page should include:

  • <header> with calculator title
  • <main>
  • A calculator container (ex: <div class="calculator">)
Hint 3: Display

Create a display element:

  • Use a <div> or <input readonly>
  • Give it an id like display
  • Default value should be 0
Hint 4: Buttons

Create buttons for:

  • Numbers 0–9
  • Operators: + − × ÷
  • = and C

Tip: use data-value attributes on buttons.

Hint 5: JavaScript variables

You will need variables for:

  • current number
  • previous number
  • current operator
Hint 6: Number button logic

When a number button is clicked:

  • Add the number to the current value
  • Update the display
  • Prevent leading zeros
Hint 7: Operator logic

When an operator is clicked:

  • Store the current number
  • Store the selected operator
  • Clear the display for the next number
Hint 8: Equals (=) logic

When = is clicked:

  • Use stored numbers and operator
  • Perform the calculation
  • Show result on the display
Hint 9: Clear (C) logic

Reset everything:

  • Display back to 0
  • Clear stored numbers
  • Clear operator
Hint 10: Division by zero

If user tries to divide by zero:

  • Show Error on the display
  • Disable further calculation until cleared
Hint 11: CSS layout
  • Center the calculator on the page
  • Use CSS grid for buttons
  • Buttons should be same size
  • Display text aligned to the right

Tip: Build UI first (HTML/CSS). Then implement logic in JavaScript step-by-step.
