Project: Blog Top Page

Create a blog homepage that follows the layout shown in the design mockup below. Match the structure, section order, and layout.

Design Mockup (SVG)

Layout order: Header → Main (Posts) + Sidebar → Footer
Desktop: 2 columns / Mobile: 1 column

1. Required Files

  • index.html (blog top page)
  • style.css (all styling)
  • assets/ (post thumbnails)
  • Optional: post.html

2. Blog Post Requirements

  • Show at least 6 posts
  • Each post must include:
    • Thumbnail image
    • Title (clickable)
    • Date
    • Category
    • Short excerpt (1–2 lines)

3. Sidebar Requirements

  • Search box (no JS needed)
  • Categories list (at least 5)
  • Recent posts list (at least 4)

4. Design & Technical Rules

  • Follow the mockup layout
  • Use flex or grid
  • Responsive (desktop & mobile)
  • Semantic HTML: header, main, article, aside, footer
  • No frameworks

Tip: Build structure first, then style to match the mockup.
