1) Use of Keyboard Characters

Definition A keyboard is an input device used to type letters, numbers, and symbols. “Keyboard characters” include letters (A–Z), numbers (0–9), punctuation (.,!?), and special symbols (@, #, $, %, &).

Types of Keys

  • Alphabet keys: A–Z (used to type words)
  • Number keys: 0–9 (used for numbers and symbols)
  • Function keys: F1–F12 (special functions, depends on OS/app)
  • Modifier keys: Shift, Ctrl, Alt (modify another key action)
  • Navigation keys: Arrow keys, Home, End, PageUp, PageDown
  • Editing keys: Backspace, Delete, Insert

Keyboard Parts

  • Typing area: letters, numbers, punctuation
  • Function row: F1–F12
  • Navigation cluster: arrows + Home/End/Page keys
  • Numeric keypad: fast numeric input (on full keyboard)

Laptop keyboards may not have a separate numeric keypad.

Combination Keys and Their Use

Combination keys improve speed when editing text or coding.

  • Ctrl + C = Copy
  • Ctrl + V = Paste
  • Ctrl + X = Cut
  • Ctrl + Z = Undo
  • Ctrl + S = Save
  • Ctrl + F = Find text
  • Ctrl + A = Select all
  • Alt + Tab = Switch applications
For coding: Ctrl + / often toggles comments in many editors (like VS Code).

Typing Tips (Technique)

  • Hand positioning: place fingers on home row keys (A S D F / J K L ;)
  • Body posture: sit straight, shoulders relaxed, elbows ~90°
  • Ergonomics: screen at eye level, wrists not bent too much
  • Typing speed: measured in WPM (Words Per Minute)

Using Typing Master (or similar apps)

  • Check total number of words typed per minute (WPM)
  • Check number of errors and accuracy percentage
  • Use games to increase speed while keeping accuracy
Goal example: Improve from 15 WPM → 25 WPM while keeping accuracy above 90%.

2) Definition of General Web Terms

Key Terms These terms are required before starting website development.

Definitions

  • Webpage: One page (example: Home page)
  • Website: A collection of webpages (example: school website)
  • Web browser: App to view webpages (Chrome, Edge, Firefox)
  • Text editor: App to write code (VS Code, Sublime, Notepad)
  • URL: Address of a webpage (example: https://example.com)
  • Hyperlink: Clickable link to go to another page

Small Examples

Example URL parts:

https://www.school.edu/news/index.html
|----protocol----| |domain-----| |path--------|

Hyperlink example:

<a href="https://www.google.com">Open Google</a>

In browsers, a link is usually blue + underlined by default.

Students often confuse “webpage” and “website”. Remember: website = many webpages.

3) Setting up Text Editor and Web Browser

Examples of Text Editors

  • VS Code (recommended for web dev)
  • Notepad / Notepad++
  • Sublime Text

Types and Function of Web Browsers

  • Chrome: very popular, strong dev tools
  • Firefox: good developer tools, open-source

Installing from Storage Drive vs Internet

  • Run the installer

From Internet (Online)

4) Creating a Web Page in HTML (Introduction)

What is a Website? (Description, Types, Function)

  • Description: A website is content published on the internet.
  • Types: personal blog, school site, business site, ecommerce, portfolio, news
  • Functions: information sharing, communication, selling, learning, marketing

Introduction to HTML

HTML is a markup language used to build the structure of a webpage. HTML does not add advanced design (that is CSS), and it does not add behavior (that is JavaScript).

HTML = Structure

Example: headings, paragraphs, lists, forms, tables

CSS = Design

Example: colors, layout, spacing, fonts

HTML Tag Definition, Types, and Examples

  • Tag: text inside < >, example: <p>
  • Paired tags: have opening and closing tags: <p>...</p>
  • Void tags: no closing tag: <img>, <br>, <input>

Tag Attributes

Attributes give extra information to tags (like link address, image source, etc). 

<a href="about.html" target="_blank">Open About Page</a>

<img src="images/logo.png" alt="School Logo" width="120">
Always include alt for images. It helps SEO and accessibility.

Developing First Simple Webpage (Full Example)

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first HTML page.</p>
</body>
</html>
Save as index.html → open with a browser.

HTML Comment

<!-- This is a comment. It will not show in browser. -->

5) Developing Webpage using HTML Tag Categories

This section contains many tag categories with examples (formatting, table, form, headings, lists, media, code, frames, grouping).

Heading Tags

Headings show document structure. Use h1 only once for main title.

<h1>School News</h1>
<h2>Term 1 Updates</h2>
<h3>Sports Day</h3>

Formatting Tags

<p>This is <strong>important</strong> text.</p>
<p>This is <em>emphasized</em> text.</p>
<p>This is <mark>highlighted</mark> text.</p>

List Tags

Unordered List

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered List

<ol>
  <li>Open editor</li>
  <li>Write HTML</li>
  <li>Save as .html</li>
</ol>

Table Tags

Tables show structured data (but do not use tables for page layout; use CSS instead).

<table border="1">
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Class</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>16</td>
      <td>S5A</td>
    </tr>
  </tbody>
</table>

Form Tags

Forms collect user input (login, registration, contact form).

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input id="name" name="name" type="text" required>

  <label for="email">Email:</label>
  <input id="email" name="email" type="email">

  <button type="submit">Send</button>
</form>

Media Tags

<img src="images/pic.jpg" alt="Student photo" width="200">

<audio controls>
  <source src="media/song.mp3" type="audio/mpeg">
</audio>

<video controls width="320">
  <source src="media/demo.mp4" type="video/mp4">
</video>

Code Tags

Use <code> for inline code and <pre> for code blocks.

<p>Use <code>Ctrl + S</code> to save.</p>

<pre><code>
function hello() {
  console.log("Hello");
}
</code></pre>

HTML Frame Tags (Historical / Not Recommended)

Frames (<frame>, <frameset>) are old and not recommended in modern web development. Modern websites use CSS layout and JavaScript frameworks.

Grouping Tags: div and span

<div>
  <h2>News</h2>
  <p>Today we learned HTML.</p>
</div>

<p>This is <span>inline</span> text.</p>
div = block container, span = inline container.

6) Use of Hyperlinks

Syntax

<a href="target-page.html">Go to another page</a>

Default Link Appearance

  • Default: blue + underlined
  • Visited: purple
  • Hover: changes style (depends on browser/CSS)

Types of Links (With Examples)

External Link

<a href="https://example.com" target="_blank" rel="noopener">
  Visit Example Website
</a>

Use target="_blank" to open new tab.

Internal Link

<a href="about.html">About Us</a>
<a href="contact.html">Contact</a>

Link to a Section (Anchor Link)

<a href="#seo">Jump to SEO Section</a>

<h2 id="seo">SEO</h2>

Email / Phone Links

<a href="mailto:teacher@school.edu">Email Teacher</a>
<a href="tel:+250700000000">Call Office</a>
Common mistake: wrong file name. If your file is About.html but you link to about.html, it may not work on some servers.

7) Use of HTML Graphics

Image Tag (img) - Most Common

<img src="images/banner.jpg" alt="School Banner" width="600">

Canvas Tag (Drawing with JavaScript)

Canvas is used to draw shapes using JavaScript (games, charts, drawing apps).

<canvas id="myCanvas" width="300" height="150" style="border:1px solid #ccc;"></canvas>

<script>
  const c = document.getElementById("myCanvas");
  const ctx = c.getContext("2d");

  // draw rectangle
  ctx.fillStyle = "blue";
  ctx.fillRect(20, 20, 100, 50);

  // draw text
  ctx.fillStyle = "black";
  ctx.font = "16px Arial";
  ctx.fillText("Canvas Demo", 20, 120);
</script>

SVG Tag (Vector Graphics in HTML)

SVG is scalable and stays clear even when zoomed.

<svg width="240" height="120">
  <rect x="10" y="10" width="120" height="60" fill="orange" />
  <circle cx="180" cy="40" r="30" fill="green" />
  <text x="10" y="110" font-size="14">SVG Example</text>
</svg>
Canvas = drawn by JS (pixels). SVG = shapes defined by XML (vector).

8) Managing Page Layout in HTML

Definition of Page Layout

Page layout means how content is arranged on a webpage (header, navigation, main content, footer).

Parts of a Webpage

  • Header: top area, site title/logo
  • Navigation: menu links
  • Main: main content
  • Sidebar: extra links/info
  • Footer: bottom info (copyright, contacts)

Use HTML Tags to Organize Page Layout

Modern HTML uses semantic tags like header, nav, main, section, article, footer.

<header>
  <h1>My School Website</h1>
</header>

<nav>
  <a href="index.html">Home</a>
  <a href="news.html">News</a>
  <a href="contact.html">Contact</a>
</nav>

<main>
  <section>
    <h2>Welcome</h2>
    <p>This is the main section.</p>
  </section>

  <article>
    <h2>Latest Announcement</h2>
    <p>School will start at 8:00 AM.</p>
  </article>
</main>

<footer>
  <p>© 2026 My School</p>
</footer>
HTML creates structure. For real layout (columns, spacing), use CSS.

9) Optimizing Webpage in HTML (SEO Basics)

Definition of Optimization

Optimization means improving your webpage so it loads faster, ranks better in search engines, and is easier for users to understand.

Benefits of Applying SEO

  • Better ranking on Google
  • More visitors
  • Clear structure helps both users and search engines

Important SEO Tags and Examples

1) Meta Tags

<head>
  <title>Student Portal - Home</title>
  <meta charset="UTF-8">
  <meta name="description" content="Student portal for learning HTML, CSS, and JavaScript.">
  <meta name="keywords" content="HTML, CSS, JavaScript, school, student portal">
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>

2) Title Tag

Title must be meaningful and specific:

  • Bad: <title>Home</title>
  • Good: <title>Kayonza TVET - ICT Department Home</title>

3) Heading Structure

<h1>ICT Department</h1>
<h2>Courses</h2>
<h3>Web Development</h3>

4) Image Alt Text

<img src="images/lab.jpg" alt="Students learning in computer lab" />

5) Semantic Tags

<header>...</header>
<nav>...</nav>
<main>...</main>
<article>...</article>
<footer>...</footer>
SEO rule: clear title + good headings + alt text + semantic structure.

10) Web Accessibility (Role, Validation, Standards)

What is Accessibility?

Web accessibility means making websites usable for everyone, including users with disabilities: visual impairment, hearing impairment, motor limitations, and cognitive challenges.

HTML Role (ARIA) - When Needed

Most of the time, semantic HTML is enough. But sometimes we add ARIA roles for extra meaning. 

<button aria-label="Close dialog">X</button>

<nav aria-label="Main navigation">
  <a href="index.html">Home</a>
  <a href="news.html">News</a>
</nav>

Form Accessibility (Labels)

<label for="email">Email</label>
<input id="email" name="email" type="email" required>
Accessibility best practice: every input should have a label.

Keyboard Navigation

  • Users should be able to navigate using Tab
  • Buttons and links must be reachable
  • Do not remove focus outline without replacement

Test and Standard Validation

  • Use HTML validation tools to check errors
  • Common errors: missing closing tags, wrong nesting, missing alt
<!-- Wrong nesting example (bad) -->
<p><div>Hello</div></p>

<!-- Correct structure (good) -->
<div>
  <p>Hello</p>
</div>
Accessibility + SEO often improve together because both require clear HTML structure.
