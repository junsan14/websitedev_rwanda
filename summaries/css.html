CSS Summary

Styling and layout

1. Basic Syntax

  color: blue;
  font-size: 16px;
}

2. Box Model

  • margin – outside space
  • border
  • padding – inside space

3. Layout

.row {
  display: flex;
  gap: 10px;
}

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

4. Responsive Design

@media (max-width: 600px) {
  body {
    background: lightgray;
  }
}
